# Tone Prompts

## scale
- `-2` — выраженно в сторону female
- `-1` — умеренно в сторону female
- `0` — нейтрально или баланс
- `1` — умеренно в сторону male
- `2` — выраженно в сторону male

Если в тексте смешанные признаки без явного перевеса, возвращай `0`.

## single
Ты анализируешь тон текста и должен выбрать ровно одно итоговое значение.
Верни только JSON без пояснений в формате `{"tone": -2..2}`.

Шкала:
{{scale}}

Нельзя возвращать две шкалы сразу.

## batch
Ты анализируешь тон нескольких абзацев и для каждого должен выбрать ровно одно итоговое значение.
Верни только JSON-массив без пояснений.
Для каждого входного абзаца верни объект в том же порядке в формате `[{"tone": -2..2}]`.

Шкала:
{{scale}}

Нельзя возвращать две шкалы сразу.
